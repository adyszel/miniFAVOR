Interfaces

assert 
use assertions_interface, only : assert
call assert( 2 > 1, "2 > 1")

Turn off assertions in production code by setting .true. to .false. via the preprocessor:

caf -cpp -DUSE_ASSERTIONS=.false. -c assertions_interface.f90

Doing so may eliminate any associated runtime overhead by enabling optimizing compilers to ignore the assertion procedure body during a dead-code-removal phase of optimization.
